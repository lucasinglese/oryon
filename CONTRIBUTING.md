# Contributing to Oryon

Thank you for your interest in contributing.

The full contributing guide is in the documentation:
**[oryonlib.dev/contributing/guide](https://oryonlib.dev/contributing/guide/)**

It covers the end-to-end workflow for adding a feature or target, the required tests and benchmarks, the PyO3 wrapper steps, and the documentation checklist.

---

For bugs or questions, open an issue on GitHub.
